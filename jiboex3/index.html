Jibo Test App

First, generate an arrow-filled board by setting the number of rows and columns you want and then clicking the Build button. The direction of the arrows will be randomized. Note: you may rebuild the board at any time.

Next, click on a square to select a start point and note whether the path from that point will leave the board. Use the animation controls to start/stop path traversal and verify the result.
